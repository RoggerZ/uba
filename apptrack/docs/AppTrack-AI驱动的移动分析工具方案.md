## 🔧 核心功能

### MVP功能（第1优先级）

#### 1. 智能SDK

**iOS SDK**：
```swift
// 1. 安装（CocoaPods）
pod 'AppTrack'

// 2. 初始化（AppDelegate）
import AppTrack

func application(_ application: UIApplication,
                 didFinishLaunchingWithOptions launchOptions: [UIApplication.LaunchOptionsKey: Any]?) -> Bool {
    AppTrack.initialize(apiKey: "YOUR_API_KEY")
    return true
}

// 3. 自动追踪（无需额外代码）
// - 应用启动
// - 屏幕浏览
// - 应用退出
// - 崩溃

// 4. 自定义事件（可选）
AppTrack.track("purchase_completed", properties: [
    "amount": 9.99,
    "product": "pro_plan"
])
```

**Android SDK**：
```kotlin
// 1. 安装（Gradle）
implementation 'io.apptrack:sdk:1.0.0'

// 2. 初始化（Application）
import io.apptrack.AppTrack

class MyApp : Application() {
    override fun onCreate() {
        super.onCreate()
        AppTrack.initialize(this, "YOUR_API_KEY")
    }
}

// 3. 自动追踪（无需额外代码）

// 4. 自定义事件（可选）
AppTrack.track("purchase_completed", mapOf(
    "amount" to 9.99,
    "product" to "pro_plan"
))
```

**SDK特性**：
- 体积：<100KB
- 内存：<5MB
- 电池影响：<1%
- 网络：<100KB/天
- 离线支持：自动缓存
- 隐私保护：不收集PII
