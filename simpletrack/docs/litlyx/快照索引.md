# Litlyx 快照索引

## 说明

- 这份索引是 Litlyx 调研快照的唯一正式入口。
- 正式阶段快照只引用 `snapshots/phase-*` 下的文件。
- `litlyx-reference/` 中的参考快照用于补齐那些在重复自动化访问里会偶发 loading 的页面。
- 正式快照和参考快照已遮挡登录邮箱、密码输入内容、workspace id 与侧栏账号区域。
