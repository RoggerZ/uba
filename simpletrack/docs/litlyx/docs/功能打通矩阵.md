## 状态说明

- “已验证”表示已经有页面截图、交互证据或真实事件回写支撑。
- “已验证页面态”表示入口和静态界面已确认，但会改变云端状态的动作尚未执行。
- “已验证受限态”表示已经确认当前账号存在门槛或 disabled 行为，但未越过这个门槛。

## 当前最值得继续补的缺口

1. `Reports` 正式 PDF 生成链路，需要账号权限解锁后继续跑。
2. `Marketing` 真实来源数据，可以后续用带 UTM 的真实访问补样本。
3. `Members` 的真实限制原因，最好补到“权限限制 / 套餐限制 / 接口异常”中的一个明确结论。
4. `Shareable links`、`Domain data`、`Sanitize Domains` 都属于会改变云端状态的动作，需要明确授权后再实测。
