# Phase 02 README

## Goal

记录 Litlyx 的安装入口、Script / GTM 双通道、Settings 里的安装信息复用，以及 Domains 标签页的表现。

## Preconditions

- 已登录且有一个工作区
- 当前工作区可进入 `Web` 和 `Settings`

## Snapshot Order

1. `Web` 的 Script 安装态
2. 切到 `Tag (GTM)`
3. 进入 `Settings / General`
4. 切到 `Settings / Domains`
5. 复核 `Domain data / Sanitize Domains` 治理入口

## Observation Focus

- 接入路径是否短
- 是否提供多种安装方式
- 是否在多个页面重复暴露脚本和 ID
- Domains 是纯空态、配置态，还是数据治理入口
