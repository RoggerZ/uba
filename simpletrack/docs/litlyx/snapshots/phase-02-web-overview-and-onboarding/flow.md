# Phase 02 Flow

1. 起点状态：已登录并进入当前工作区
2. 用户动作：查看 `Web` 默认安装页
3. 页面响应：展示 Script 代码、框架快捷入口、AI setup 和验证按钮，对应 `P02-S01`
4. 用户动作：切换到 `Tag (GTM)`
5. 页面响应：展示 GTM 接入代码，对应 `P02-S02`
6. 用户动作：进入 `Settings / General`
7. 页面响应：再次看到工作区名、ID、脚本代码和删除入口，对应 `P02-S03`
8. 用户动作：切到 `Settings / Domains`
9. 页面响应：早期截图只捕捉到容器和 tab，对应 `P02-S04`
10. 复核响应：Domains 实际提供 `Domain data` 和 `Sanitize Domains` 两块数据清理能力，对应 `P02-S05`
11. 设计启发：Litlyx 的安装体验强，Settings 里的 Domains 更接近治理 / 删除数据入口，操作前需要明显风险提示
