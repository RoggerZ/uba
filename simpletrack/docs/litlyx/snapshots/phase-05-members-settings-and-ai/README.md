# Phase 05 README

## Goal

记录 Litlyx 的治理层、AI 助手和套餐页，理解它如何把协作、治理、AI 和商业化串起来。

## Preconditions

- 已登录并可进入 `Shields / Analyst / Plans`

## Snapshot Order

1. `Shields` 默认域名页
2. `IP addresses`
3. `Bot traffic`
4. `Add domain` 弹窗
5. AI 助手默认态
6. AI 示例展开态
7. Plans 个人版
8. Plans 企业版
9. FAQ 展开态
10. Shareable links 空态
11. Members loading / 受限态

## Observation Focus

- 数据治理是否被单独产品化
- 分享链接是否和成员邀请分离
- AI 是否定位为分析助手
- 套餐分层是否直观影响 IA 和功能可见性
- Members 不可用时是否有清晰解释
- FAQ 是否服务于转化而不是售后
