# Phase 04 README

## Goal

记录 Marketing 模块的空态和示例态、UTM 生成器、Reports 模板中心，以及 SEO 付费门槛。

## Preconditions

- 已登录并可进入 `Marketing / Reports / SEO`

## Snapshot Order

1. Marketing 默认空态
2. Marketing 示例数据态
3. `Generate UTM link` 弹窗
4. Reports 模板页
5. Reports `Sample` PDF 预览态
6. Reports 正式生成禁用态
7. SEO 门槛页

## Observation Focus

- UTM 是否和分析页形成闭环
- 示例数据是否让 Marketing 更容易理解
- Reports 是模板中心还是简单导出
- Sample 预览是否独立于正式生成权限
- 当前账户能否进入正式报告生成流程
- SEO 的付费门槛是否清晰
