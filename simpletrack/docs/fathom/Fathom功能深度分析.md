### 2. 事件追踪和转化

#### 2.1 自定义事件（Event Tracking）

**设置方式**：

**方法1：JavaScript API**
```javascript
// 基础事件追踪
fathom.trackEvent('Signup')

// 带金额的事件（收入追踪）
fathom.trackEvent('Purchase', {_value: 9900}) // 单位：分

// 自定义事件ID
fathom.trackEvent('Button Click', {_site_id: 'YOUR_SITE_ID'})
```

**方法2：HTML属性（无代码）**
```html
<!-- 自动追踪点击 -->
<button data-fathom="signup">注册</button>

<!-- 追踪链接点击 -->
<a href="/pricing" data-fathom="view-pricing">查看定价</a>
```

**支持的事件类型**：
```
1. 自定义事件（Custom Events）
   - 按钮点击
   - 表单提交
   - 视频播放
   - 任何用户交互

2. 页面浏览目标（Pageview Goals）
   - 追踪特定页面访问
   - 如：/thank-you, /success

3. 收入追踪（Revenue Tracking）
   - 追踪交易金额
   - 电商收入分析
```
