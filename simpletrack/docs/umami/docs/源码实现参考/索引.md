## 给 SimpleTrack 的启发

SimpleTrack 的文档、官网和产品空态应把这些数据点组织成用户能复验的链路：安装、发送、Realtime 查看、Events 排障。索引页可以作为 PRD 或研发任务拆分时的技术依据。

## 给 analytics-core 的启发

`analytics-core` 的接口和测试可以按这个索引逐项对齐：collect 请求字段、标准事件 envelope、事件属性展开、ClickHouse 写入、Realtime 查询、Events 查询和 filter whitelist 都应有明确验收。
