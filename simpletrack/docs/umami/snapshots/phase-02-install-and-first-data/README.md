# Phase 02 README

## Goal

记录 Website settings、Tracking Code、Share、Danger zone，以及 demo 的接入过程。

## Preconditions

- 已创建 website
- 已拿到 `website id`
- `tracking-demo/` 静态服务已经启动

## Snapshot Order

1. 打开 Overview 空态
2. 打开 settings 全页
3. 分别记录 Website basics / Tracking code / Share / Danger zone
4. 启动本地 demo 页面
5. 加载 tracker
6. 回到 Overview 观察是否出现首批数据

## Observation Focus

- 安装说明是否足够短
- tracker 接入阻力
- settings 是否被合理拆块
- 首批数据的反馈速度
