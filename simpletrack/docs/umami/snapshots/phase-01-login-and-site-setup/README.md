# Phase 01 README

## Goal

记录 Umami Cloud 的登录、站点列表或首次建站入口、创建网站表单，以及创建后进入站点的路径。

## Preconditions

- 已具备 Umami Cloud 账号
- 浏览器已可访问 `https://cloud.umami.is`

## Snapshot Order

1. 打开登录页
2. 完成登录
3. 进入站点列表或首次建站入口
4. 打开创建网站表单
5. 完成创建并进入站点

## Observation Focus

- 登录入口信息密度
- 首屏站点入口是否清晰
- 创建网站表单字段量
- 创建后回到站点的路径长度
