# Phase 01 Flow

1. 起点状态：未登录 Umami Cloud
2. 用户动作：打开登录页并输入账号信息
3. 页面响应：登录页、已填表单、登录后 `Websites` 列表页依次出现
4. 截图编号：`P01-S01` `P01-S02` `P01-S03`
5. 用户动作：点击 `Add website`
6. 页面响应：弹出只包含 `Name / Domain` 的建站弹窗
7. 截图编号：`P01-S04` `P01-S05`
8. 用户动作：保存 `simpletrack-local / localhost`
9. 页面响应：新站点出现在列表中
10. 截图编号：`P01-S06`
11. 对产品设计的启发：SimpleTrack 的首次建站流可以直接借鉴这种最小字段弹窗
