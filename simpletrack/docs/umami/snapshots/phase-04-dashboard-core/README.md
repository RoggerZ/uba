# Phase 04 README

## Goal

把 Umami 的 `Dashboard` 和 `Boards` 拆开看清楚，避免把两套入口混为一谈。

## Preconditions

- 已登录 Cloud
- 账号具备查看 Dashboard 和 Boards 的权限

## Snapshot Order

1. 打开全局 `Dashboard`
2. 打开 `Edit Dashboard`
3. 打开 `Boards` 列表
4. 打开 `Add board`
5. 展开 `Board type`

## Observation Focus

- Dashboard 和 Boards 的边界
- Dashboard 是否更像“全局默认看板”
- Boards 是否更像“用户自建看板系统”
- Board type 是否是比 component 更高一级的抽象
