# Phase 04 Flow

1. 起点状态：已登录 Umami Cloud
2. 用户动作：打开顶层 `Dashboard`
3. 页面响应：出现全局 dashboard 视图和 `Design` 入口
4. 截图编号：`P04-D00`
5. 用户动作：进入 `Edit Dashboard`
6. 页面响应：出现全局 dashboard 的编辑态
7. 截图编号：`P04-D04`
8. 用户动作：切换到 `Boards`
9. 页面响应：出现 board 列表与 `Add board`
10. 截图编号：`P04-D01`
11. 用户动作：点击 `Add board` 并展开 `Board type`
12. 页面响应：出现 `Mixed / Website / Pixel / Link`
13. 截图编号：`P04-D02` `P04-D03`
14. 对产品设计的启发：Dashboard 更像系统级默认看板，Boards 更像用户可配置看板工厂，二者不应混用
